ADD YOUR OWN JUPYTER NOTEBOOKS

To begin creating your own Jupyter notebook, visit the central GitHub repository
and ‘fork’ your own copy of the platform. Once complete, it is possible to start
adding notebooks and papers.

After converting a notebook into html (either externally or on JupyterLab), add the
file into the notebooks folder. following this, add an entry into the manifest.json
file.

This is also where the url to the paper is added and the explanation. 
Although it is intended that papers be presented via sites without paywalls or
institutional requirements (such as arXiv), it is still possible to provide links that
require payment or institutions to access

Once these elements have been added, select commit changes, and the website shall be published.
Researchers can provide as many notebooks as necessary as long
as the manifest.json folder is updated accordingly.
