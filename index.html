Fast export & embed

Export Jupyter notebooks to static HTML with nbconvert, and embed them on this site for easy sharing and presentation.

How it works

  1. Upload notebooks (.ipynb) to the repository.
  2. GitHub Actions will export them to static HTML.
  3. Open the Notebooks page to see the embedded outputs.
