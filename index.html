Fast export + embed

Export notebooks to static HTML with nbconvert, and this site will embed them for easy sharing and presentation.

How it works

  1. Upload notebooks (.ipynb) to notebooks_src/ via the repo web UI.
  2. Push / commit — GitHub Actions will run and export them into docs/notebooks/.
  3. Open the Notebooks page to see each exported notebook embedded with outputs.
